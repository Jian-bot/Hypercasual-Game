# Hypercasual-Game
**3D Hypercasual Game series containing two fun and engaging games.**
* 3D Color Switch (Jian)
  - Player can hit part of obstacles that has the same color as the ball, otherwise explodes. 
* Pong (Ryan)
  - Two-player air hockey game.
* Game Scene build by level:  -> Assets/Scenes
* Scripts for component control: ->Assets/script

## 3D Color Switch UI
**Right clicked to hold the ball during game**
* **QUICK START:** Start from level 0.
* **MAIN:** Back to MainScreen UI.
* **QUIT:** Quit game.
* **LEVELS:** Select your favorite level to start the game. 
<img width="837" alt="ColorSwitch_UI" src="https://github.com/Jian-bot/Hypercasual-Game/assets/76824957/a87faee8-3763-4b1e-aa82-f21239574232">

## Game Lost 
* **Restart:** Restart the current level of game.
* **Home:** Back to 3D Color Switch UI.
